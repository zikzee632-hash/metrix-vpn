# MetRix VPN

Fast & Secure VPN app built with Capacitor.

## Setup

```bash
npm install
npx cap add android
npx cap sync android
```

## Build APK

Push to GitHub — Actions will build automatically.
Or locally:
```bash
cd android && ./gradlew assembleDebug
```

APK: `android/app/build/outputs/apk/debug/app-debug.apk`
